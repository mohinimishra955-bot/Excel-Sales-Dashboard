# Sales Performance Dashboard 📊

An interactive Excel dashboard analyzing sales data across regions, products, and months.

## Features
- Region-wise sales analysis
- Product-wise profit breakdown  
- Monthly sales trends
- Interactive slicers and charts
- Key insights and recommendations

## Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Charts & Graphs

## Key Insights
- Central region generated highest sales
- March was the best performing month
- Smartphone delivered highest profit
